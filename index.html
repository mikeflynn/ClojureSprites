ClojureSprites

A Clojure application to automatically create and implement a sprite from a CSS file.

Usage

./bin/clojuresprite --incss=/input/css/file.css --outsprite=/output/img/sprite.png [--outcss=/output/css/file.css] [--tempdir=/tmp/whatever]
